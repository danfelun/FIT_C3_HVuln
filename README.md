## 🎯 Descripción

Este laboratorio permite a los estudiantes ejecutar un flujo completo de **identificación, análisis y validación de vulnerabilidades** sobre una máquina virtual vulnerable previamente importada (OVA).

---

## 🧰 Herramientas incluidas

- Nmap
- Nikto
- Nuclei
- OWASP ZAP
- Metasploit

---

## 🚀 Despliegue

```bash
git clone https://github.com/danfelun/FIT_C3_HVuln.git
cd FIT_C3_HVuln
chmod +x lab.sh
docker compose up -d
```

---

## 🧪 Uso

```bash
./lab.sh
```
